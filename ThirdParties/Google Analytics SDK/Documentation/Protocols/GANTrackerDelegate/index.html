GANTrackerDelegate

Google Analytics tracker delegate. This delegate gets notified when the tracker dispatches a hit successfully, or when there was an error dispatching the hit. All delegate methods are optional.

Extends Protocol: NSObject
Declared In: GANTracker.h


Methods

-hitDispatched:

This method is invoked when a hit has been successfully submitted to Google Analytics.

-trackerDispatchDidComplete:eventsDispatched:eventsFailedDispatch:

This method is invoked when a dispatch to Google Analytics is completed.

hitDispatched:

This method is invoked when a hit has been successfully submitted to Google Analytics. 

- (void)hitDispatched:(NSString *)hitString;
Parameters
hitString

The string representation of the hit, as sent to Google Analytics.

trackerDispatchDidComplete:eventsDispatched:eventsFailedDispatch:

This method is invoked when a dispatch to Google Analytics is completed. 

- (void)trackerDispatchDidComplete:(GANTracker *)tracker eventsDispatched:(NSUInteger)hitsDispatched eventsFailedDispatch:(NSUInteger)hitsFailedDispatch;
Parameters
tracker

The tracker object for which the dispatch was made.

hitsDispatched

The number of hits successfully submitted to Google Analytics.

hitsFailedDispatch

The number of hits that could not be submitted.

Discussion

Note: failed hits will be retried on the next dispatch.
